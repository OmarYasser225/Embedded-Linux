# Back Trace
This project is used to trace the flow of code which display the all function called in stack and display them when they finsihed as the following

```
Enter to [main]
Enter to [fun1]
Enter to [fun2]
Enter to [fun3]
-------------------------------------------------------------------
Backtrace as follows:
0- fun3
1- fun2
2- fun1
3- main
Back Trace is Finished
-------------------------------------------------------------------
Exit From [fun3]
Exit From [fun2]
Exit From [fun1]
Exit From [main]
```
